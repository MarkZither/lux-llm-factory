# Security Policy

## Reporting a Vulnerability

Please report suspected vulnerabilities by email:

- security@zither-it.co.uk

Do not open public issues for undisclosed vulnerabilities.

## Response Process

Security reports are triaged privately.

- Initial acknowledgement: TBD
- Status updates: provided as investigation progresses
- Resolution and disclosure: coordinated after remediation is available

## Scope

This policy applies to all code and documentation in this repository.

## Responsible Disclosure

Please provide enough detail to reproduce the issue.
Allow reasonable time for investigation and remediation before public disclosure.
